# 需求文档 — RemovebgStranger（AI 涂抹去除人物 · 智能背景修复）

> 最后更新：2026-03-31（第二次更新）

---

## 一、项目背景

用户上传照片后，通过画笔涂抹的方式手动标记要去除的人物区域，系统使用 AI inpainting 模型（LaMa）智能修复被涂抹区域的背景，最终效果为路人"凭空消失"，背景自然融合。UI 简洁现代，参考 remove.bg 风格，支持中英双语切换。

---

## 二、目标用户

- 需要从照片中去除路人/闯入者的普通用户
- 摄影后期及内容创作者
- 社交媒体、营销短视频相关人员

---

## 三、核心需求（当前实现）

### 1. 上传功能
- 支持 JPG、PNG、WebP 格式
- 支持点击选择或拖拽上传
- 上传后立即进入涂抹阶段（无等待）

### 2. 橡皮擦涂抹工具（替代原 YOLO 检测方案）
- 用画笔在图片上涂抹要去除的人物
- 可调节画笔大小（10px ~ 120px）
- 支持"清除重画"重置涂抹区域
- 支持触屏操作（移动端）
- 涂抹区域以半透明红色实时显示

### 3. AI 背景修复（inpainting）
- 使用 iopaint **MAT（Mask-Aware Transformer）** 模型（2026-03-31 从 LaMa 升级）
- MAT 具备全局语义理解，幻影/重复纹理问题明显优于 LaMa
- 采用 CROP 策略，大图分块处理减少拼接痕迹
- 超过 1200px 的图片自动缩小处理后放大回原尺寸
- 涂抹区域自动膨胀处理，覆盖人物边缘（含发丝/阴影）
- 未涂抹区域背景像素完全保留（手动合并）

### 4. 结果展示及下载
- 处理完成后并排显示原图 vs 结果图
- 支持一键下载 PNG 格式结果图
- **🖌️ Refine Again（继续修复）按钮**（2026-03-31 新增）：将当前结果图作为新输入，跳回涂抹阶段，用户可对残影区域重新涂抹再次修复，支持无限迭代

### 5. 国际化（i18n）
- 支持中文 / English 双语切换
- 右上角语言切换按钮：**EN 在左，中文 在右**（2026-03-31 调整）
- **默认语言：英文**（2026-03-31 调整，原为中文）
- 所有页面（上传、涂抹、结果）全部适配

### 6. UI 设计
- 简洁现代，蓝紫渐变风格
- 三步进度条导航（Upload → Paint → Download）
- 响应式布局，支持移动端
- 无存储，隐私保护提示

---

## 五、当前流程

1. 用户上传图片 → 立即进入涂抹页
2. 用画笔涂抹要去除的人物（红色半透明笔迹）
3. 点击"Remove Painted Area"→ 前端生成黑底白色 mask PNG 发送至后端
4. 后端调用 iopaint LaMa 对涂抹区域做 inpainting
5. 返回修复后的 PNG，前端展示对比图
6. 用户下载结果

---

## 六、已知限制

- **CPU 处理慢**：无 GPU，LaMa 处理约 30-60 秒
- **大面积遮挡效果一般**：LaMa 对大面积（人物占图 30%+）修复效果差于 GPU 模型
- **API 超时**：已设置 300 秒超时，但极大图片仍有风险

cd /root/projects/RemovebgStranger && npm run dev
