# MVP需求文档 — AI智能背景去除（用户可选择去除指定人物，保留背景原貌）

## 一、项目背景  
用户上传照片后，系统自动检测所有人物，用户通过点击或圈选指定要去除的人物，系统仅让选中人物在最终图像中"隐形"或透明化，背景完全保留原始像素，无任何AI或图像修复填充，保证背景真实不变。UI设计简洁现代，参考 remove.bg，部署于 Cloudflare，无存储。

## 二、目标用户  
- 需要智能去除指定人物背景的普通用户  
- 摄影后期及内容创作者  
- 社交媒体、营销短视频相关人员  

## 三、核心需求  
1. **上传功能**  
   - 支持JPG、PNG等主流图片格式  
   - 界面简洁，支持点击或拖拽上传

2. **人物实例检测与分割**  
   - 自动检测图片中的所有人物实例  
   - 生成每个人物对应的掩码（mask）

3. **用户交互选择**  
   - 支持点击单个人物选择  
   - 支持圈选多个人物区域选择  
   - 可取消及调整选择状态

4. **去除指定人物**  
   - 仅将用户选中的人物区域像素置为透明  
   - **不对背景像素进行更改、不执行AI修复填充**  
   - 输出支持带透明通道的PNG格式图片

5. **结果展示及下载**  
   - 处理完成后在页面预览带透明通道的图片  
   - 支持用户下载该透明背景结果图

6. **UI设计**  
   - 简洁现代，参照 remove.bg 风格  
   - 清晰突出操作区域与结果  

7. **部署方案**  
   - 部署于 Cloudflare Worker 或 Cloudflare Pages  
   - 无持久化存储，确保隐私安全  

## 四、技术要点  
- 使用实例分割模型（如 Detectron2、YOLOv8-seg）提取人物掩码  
- 将用户交互选择映射至对应掩码，定位要去除的人物  
- 对选中掩码区域做透明处理，未选区域保持不变  
- 前端利用Canvas实现交互选区和结果展示  
- 后端无状态API设计，支持高并发

## 五、流程说明  
1. 用户上传图片  
2. 后端返回所有人物掩码及坐标信息  
3. 用户点击或圈选选中人物  
4. 向后端提交选中人物掩码索引  
5. 后端返回处理后PNG图（透明选中人物区域）  
6. 前端展示结果，用户可下载

## 六、MVP范围界定  
- 单张图片处理  
- 仅支持人物实例分割  
- 简单选区工具，无复杂边缘细化  
- 不进行背景修复或AI生成填充  

## 七、后续扩展方向（非MVP）  
- 支持多图批量处理  
- 增加复杂边缘调整工具  
- 支持视频及动态图像处理  
- 引入智能边缘优化或图像增强
